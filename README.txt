Theo — 아트 후원 플랫폼 (디자인 시안)
========================================

포함 파일
- Theo 사이트.html      : 4개 페이지 통합 사이트 (About / 아티스트 / 스토리 / 후원내역)
- Theo 시안비교.html    : 둘러보기 페이지 3가지 레이아웃 시안 비교 + 모바일 프레임

여는 방법
- 각 .html 파일을 더블클릭하면 웹브라우저에서 바로 열립니다.
- 별도 설치 없이 단일 파일로 동작합니다.

기능
- 상단 메뉴/탭 클릭으로 페이지 전환
- 컬러 테마 3종(선셋 · 페리윙클 · 세이지) 스와치 전환
- 라이트 / 다크 모드 토글
- 카테고리 필터 · 정렬 · 좋아요 · 팔로우 등 인터랙션 동작

참고
- 작품/작가 썸네일은 공용 placeholder 사진(picsum.photos)을 사용하므로
  표시하려면 인터넷 연결이 필요합니다. 폰트와 스타일은 파일에 내장되어 있습니다.
- 모든 내용은 더미 데이터로 구성된 디자인 시안입니다.
